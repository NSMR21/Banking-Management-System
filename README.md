# Banking Management System (C)

A console-based banking application developed in C that allows users to perform essential banking operations such as account creation, transactions, and record management with persistent storage.


## Features

- Create a new bank account  
- Delete existing account  
- Deposit money  
- Withdraw money  
- Check account balance  
- Store and retrieve data using file handling  
- Menu-driven user interface  


## Tech Stack

- Language: C  
- Concepts Used:
  - File Handling
  - Structures
  - Functions (Modular Programming)
  - Input Validation
  - Basic Data Management  


## How to Run

1. Clone the repository:
   git clone https://github.com/NSMR21/Banking-Management-System.git

2. Navigate to the project directory:
   cd Banking-Management-System

3. Compile the program:
   gcc BankManagementSystem.c -o bank

4. Run the program:
   ./bank


## Required Files

Make sure the following files are present in the same directory:

accounts.txt → stores account details
transactions.txt → stores transaction records
