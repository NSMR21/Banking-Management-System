# Banking Management System (C)

A console-based banking application developed in C that allows users to perform essential banking operations such as account creation, transactions, and record management with persistent storage.


## Features

- Create a new bank account  
- Delete existing account  
- Deposit money  
- Withdraw money  
- Check account balance  
- Store and retrieve data using file handling  
- Menu-driven user interface  


## Tech Stack

- Language: C  
- Concepts Used:
  - File Handling
  - Structures
  - Functions (Modular Programming)
  - Input Validation
  - Basic Data Management
